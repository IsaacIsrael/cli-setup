# Roadmap

`cli-setup` grows one capability at a time. Here is what is planned for you to
use, in order:

| Capability | What you will be able to do |
| --- | --- |
| Install | Get `cli-setup` onto your machine with a single command |
| `doctor mobile` | Diagnose a mobile (React Native) environment — report what is missing, changing nothing |
| `setup mobile` | Install the missing tools for the mobile profile, idempotently |
| `setup mobile` (GUI tools) | Guided setup for tools with a graphical step, like Xcode and Android Studio |
| Team config + `update` / `config` | Share a team standard and reconcile your tools to it |

For the authoritative, always-current plan, see the
[issue tracker](https://github.com/IsaacIsrael/cli-setup/issues).
